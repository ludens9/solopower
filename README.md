# 🏠 1인 가구 생존 유형 테스트

바이럴 웹 페이지로 제작된 1인 가구 생존 유형 테스트입니다. JSON 파일을 수정하면 자동으로 웹페이지에 반영됩니다.

## 📁 파일 구조

```
solopower/
├── index.html          # 메인 HTML 파일
├── styles.css          # CSS 스타일 파일
├── script.js           # JavaScript 로직 파일
├── result.json         # 결과 페이지 콘텐츠 (JSON 형식)
├── plan.md             # 기획 및 디자인 문서
└── README.md           # 사용법 안내
```

## 🚀 사용법

### 1. 기본 실행
1. 모든 파일을 같은 폴더에 저장
2. `index.html` 파일을 웹브라우저에서 열기
3. 테스트 시작!

### 2. 콘텐츠 수정하기
비개발자도 쉽게 수정할 수 있습니다:

#### 결과 수정하기
`result.json` 파일에서 각 유형의 결과를 수정하세요:
```json
{
  "economic_stable_life_stable_relationship_yes_emotion_satisfied": {
    "animal": "🐘",
    "title": "코끼리 — 완벽 생존자형",
    "current": "여기에 설명을 수정하면...",
    "solution": "여기에 솔루션을 수정하면...",
    "broccoli": ["수정된 항목1", "수정된 항목2"]
  }
}
```

### 3. 자동 반영 확인
- JSON 파일을 수정한 후 웹페이지를 새로고침하면 변경사항이 즉시 반영됩니다
- 개발자 도구 콘솔에서 "JSON 파일에서 결과 데이터를 성공적으로 로드했습니다" 메시지 확인

## 🎨 디자인 특징

- **모바일 퍼스트**: 모바일에서 최적화된 사용자 경험
- **반응형 디자인**: 다양한 화면 크기에 대응
- **직관적인 UI**: 카드 선택형 인터페이스
- **부드러운 애니메이션**: 선택 시 하이라이트 및 전환 효과
- **접근성 고려**: 키보드 네비게이션 및 스크린 리더 지원

## 🔧 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **Vanilla JavaScript**: 프레임워크 없이 순수 JS
- **JSON**: 구조화된 데이터 형식

## 📱 지원 브라우저

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 🎯 주요 기능

1. **8단계 질문**: 4개 축(경제, 생활, 관계, 정서) 평가
2. **16가지 결과 유형**: 동물 캐릭터로 표현된 생존 유형
3. **실시간 진행 표시**: 진행 바로 현재 단계 확인
4. **결과 공유**: 카카오톡, 인스타그램 등으로 공유 가능
5. **자동 JSON 반영**: 파일 수정 시 자동으로 웹페이지 업데이트

## 🛠️ 개발자 정보

- **서버 불필요**: 정적 파일만으로 동작
- **CDN 없음**: 모든 리소스 로컬 저장
- **간단한 구조**: 비개발자도 쉽게 이해하고 수정 가능
- **JSON 기반**: 안정적이고 직관적인 데이터 관리
